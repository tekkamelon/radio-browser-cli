# 応答のルール

- 日本語

- 記号は可能な限り半角文字を使用

# 調査の際の指示

- 調査する際は複数の言語で調査して構いません

- 明示されない限りインターネットで調査して構いません

- 明示されない限り調査結果を`markdown`で出力してください

- シェルコマンド実行の際の指示

- 与えられたタスクの終了後

    - 以下のコマンドを実行してください

        - `notify-send -t 3000 -u normal 'AGENTS' 'タスク完了' -i ; paplay /usr/share/sounds/freedesktop/stereo/complete.oga`

            - もし実行できない場合はそのまま終了して構いません

# コードの書式

- 差分ごとにコードを出力

- ソースコード本体には行番号を付与しない

- コメント

    - 記号は可能な限り半角文字を使用

- インデントはタブ
